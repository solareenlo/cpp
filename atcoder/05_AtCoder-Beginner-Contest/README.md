- [kyuridenamida/atcoder-tools](https://github.com/kyuridenamida/atcoder-tools) で解答の雛形を自動生成しています．

### Usage
```bash
atcoder-tools help
atcoder-tools gen abc001
cd abc001/A
vim main.cpp
g++ main.cpp
atcoder-tools test
atcoder-tools submit
```
